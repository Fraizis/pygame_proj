## Игра 'Змейка' на python в стиле ООП
Классическая игра "Змейка" написанная на python с помощью библиотеки pygame, 
с подсчетом очков
### Управление
Змейка управляется стрелочками на клавиатуре 
* ESC - новая игра 
* ENTER - начать игру 
* SPACE - пауза 

### Установка и запуск
Находясь в корневом каталоге проекта установить зависимости командой

    pip install -r requirements.txt

Запустить игру:
    
    запустить модуль main.py

либо находясь в папке с проектом ввести в терминале 

    python3 main.py
